# Extension Icons

Create the following icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

Suggested design: A brain icon or mind map nodes.

You can use tools like Figma, Canva, or online icon generators.
For development, you can use placeholder icons from https://placehold.co
